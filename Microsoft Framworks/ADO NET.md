Framework to interact with [[Databases]] using [[C Sharp]] in [[NET Core]]. 

[[Entity Framework Core]] is built on top of ADO .NET as a framework for using making using ADO NET easier. 

ADO NET can also be used to interact with data from other sources like XML files.
